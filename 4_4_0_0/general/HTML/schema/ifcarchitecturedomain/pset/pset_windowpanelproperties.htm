7.1.4.5 Pset_WindowPanelProperties

PSET_TYPEDRIVENOVERRIDE / IfcPlate,IfcWindow

Properties of the window panel.
HISTORY  New property set in IFC4.3.2.0 to replace the entity IfcWindowPanelProperties
Properties

PSD-XML

NameTypeDescription
OperationType P_ENUMERATEDVALUE / IfcLabel / PEnum_WindowPanelOperationEnum Type of operations. Also used to assign standard symbolic presentations according to national building standards. For a window, it is the type of window panel operations. Also used to assign standard symbolic presentations according to national building standards.
PanelPosition P_ENUMERATEDVALUE / IfcLabel / PEnum_WindowPanelPositionEnum Position of the panel. For a window, it is the position of the panel within the overall window style.
FrameDepth P_SINGLEVALUE / IfcPositiveLengthMeasure The length (or depth) of the frame. For a window, it is the depth of panel frame, measured from front face to back face horizontally (i.e. perpendicular to the window elevation plane).
FrameThickness P_SINGLEVALUE / IfcPositiveLengthMeasure The thickness of the frame. For a window, it is the width of panel frame, measured from inside of panel (at glazing) to outside of panel (at lining), i.e. parallel to the window (elevation) plane.

6.5.4.49 Pset_SpatialGeoObsBorehole

PSET_OCCURRENCEDRIVEN / IfcBorehole

Properties for geoscience observations in boreholes.
Change log
ItemSPFXMLChangeDescription
IFC4X3_ADD2 to IFC4X4_draft
    Pset_SpatialGeoObsBoreholeADDED
Properties
NameTypeDescription
BoreholeElevation P_SINGLEVALUE / IfcLengthMeasure If present, the property reports the elevation data in meters for the borehole (i.e., wellbore) start point. This is a compromise approach to allow for delivery of legacy 2D data without elevation data, and for software that cannot process a 3D GM_Point.
BoreholeInclination P_SINGLEVALUE / IfcPlaneAngleMeasure The angle formed by a vertical line and the vertical component in the borehole direction is entered.
BoreholeAzimuth P_SINGLEVALUE / IfcPlaneAngleMeasure If the drilling is inclined, the horizontal component in the borehole direction is entered based on the 360 degree orientation method moving right from true north.
BoreholePurpose P_SINGLEVALUE / IfcLabel The purpose property reports the purpose for which the borehole was drilled.
Operator P_SINGLEVALUE / IfcLabel The organisation or agency responsible for commissioning the borehole (as opposed to the agency which drilled the borehole).
DrillingContractor P_SINGLEVALUE / IfcLabel The property DrillingContractor reports the organisation responsible for drilling the borehole (as opposed to commissioning the borehole).
BoreholeLengthPlanned P_SINGLEVALUE / IfcPositiveLengthMeasure The planned length of a borehole e.g. in a site investigation program, in metres, as determined by the data provider.
BoreholeLengthDrilled P_SINGLEVALUE / IfcPositiveLengthMeasure The length of a borehole as drilled and logged, in metres, as determined by the data provider.
DrillingMethod P_SINGLEVALUE / IfcLabel Type of drilling methods used to perform the investigation.

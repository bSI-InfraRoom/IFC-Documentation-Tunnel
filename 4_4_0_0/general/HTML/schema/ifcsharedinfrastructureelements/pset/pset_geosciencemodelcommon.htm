6.5.4.33 Pset_GeoScienceModelCommon

PSET_OCCURRENCEDRIVEN / IfcGeoScienceModel

Common properties for geoscience models.
Change log
ItemSPFXMLChangeDescription
IFC4X3_ADD2 to IFC4X4_draft
    Pset_GeoScienceModelCommonADDED
Properties
NameTypeDescription
ModelUseCase P_SINGLEVALUE / IfcText Specifies the use case for which the model was created. ModelUseCase could be e.g.: stability assessment, quantity assessment, tendering etc.
CreationDate P_SINGLEVALUE / IfcDate Creation date of the model.
ReleaseDate P_SINGLEVALUE / IfcDate Date of release of the model.
UpdateDate P_SINGLEVALUE / IfcDate Date of update of the model.
VersionNumber P_SINGLEVALUE / IfcLabel Version tag of the model regarding its descriptive informations.
ModelPurpose P_SINGLEVALUE / IfcLabel Purpose for which the model was done. Purpose could be e.g.: scientific study, mining, oil reservoir, geotechnics, hydrogeology, simulation or a URI from the codelist https://data.geoscience.earth/ncl/ModelPurpose.
ModelStatus P_SINGLEVALUE / IfcLabel Status of the model. Status could be e.g.: ended, in-progress, in-development or a URI from the codelist https://data.geoscience.earth/ncl/ModelStatus.
Authors P_LISTVALUE / IfcLabel Name of the author(s) involved in producing the model.
Organizations P_LISTVALUE / IfcLabel Name of the organization(s) involved in producing the model.
AreaOfInterest P_SINGLEVALUE / IfcIdentifier Reference to geographical gazeetteer entries (France, Paris basin etc).
NOTE Geonames (https://www.geonames.org/) could be used in case no other is available.

ProfileType P_SINGLEVALUE / IfcIdentifier Contains a term from a controlled vocabulary specifying the concave/convex geometry of fold relative to earth surface, and relationship to younging direction in folded strata if known (e.g., antiform, synform, neutral, anticline, syncline, monocline, ptygmatic).
StrikeDirection P_SINGLEVALUE / IfcPlaneAngleMeasure The geographic direction of a line created by the intersection of a plane and the horizontal plane. If non specific convention is used, this angular value is in the range 0 to 180 degrees. A strike value is generally completed with an indication of the dip direction.
DipDirection P_SINGLEVALUE / IfcPlaneAngleMeasure The geographic direction of a line created by the intersection of a plane and the horizontal plane. If non specific convention is used, this angular value is in the range 0 to 180 degrees. In this case, there is an ambiguity on the dip orientation. For example, a plane with an orientation of 90 degrees from the north could either have a dip direction to the North or to the South. Thus, to avoid this ambiguity, a strike value is generally completed with an indication of the dip orientation (Quadrant).

6.3.4.11 Pset_FastenerWeld

PSET_TYPEDRIVENOVERRIDE / IfcFastener / WELD

Fastener WeldProperties related to welded connections.
Properties
NameTypeDescription
Type1 P_SINGLEVALUE / IfcLabel
Type1Type of weld seam according to ISO 2553. Note, combined welds are given by two corresponding symbols in the direction of the normal axis of the coordinate system. For example, an X weld is specified by Type1 = 'V' and Type2 = 'V'.
Type2 P_SINGLEVALUE / IfcLabel
Type2See Type1.
Surface1 P_SINGLEVALUE / IfcLabel
Surface1Aspect of weld seam surface, i.e. 'plane', 'curved' or 'hollow'. Combined welds are given by two corresponding symbols analogous to Type1 and Type2.
Surface2 P_SINGLEVALUE / IfcLabel
Surface2See Surface1.
Process P_SINGLEVALUE / IfcInteger
ProcessReference number of the welding process according to ISO 4063, an up to three digits long code
ProcessName P_SINGLEVALUE / IfcLabel
Process NameName of the welding process. Alternative to the numeric Process property.
NominalThroatThickness P_SINGLEVALUE / IfcPositiveLengthMeasure
aDesign value of the height of the largest isosceles triangle that can be inscribed in the section of a fillet weld.
REFERENCE  Symbol a according to ISO 2553:2019.
WeldWidth P_SINGLEVALUE / IfcPositiveLengthMeasure
cRequired elongated hole width at the faying surface or seam weld width at the faying surface.
REFERENCE  Symbol c according to ISO 2553:2019.
WeldDiameter P_SINGLEVALUE / IfcPositiveLengthMeasure
dDimension of the required hole diameter at the faying surface, or required spot weld diameter at the faying surface, or required stud diameter.
REFERENCE  Symbol d according to ISO 2553:2019.
WeldElementSpacing P_SINGLEVALUE / IfcPositiveLengthMeasure
eSpacing between weld elements (centre to centre)
REFERENCE  Symbol e according to ISO 2553:2019.
WeldElementLength P_SINGLEVALUE / IfcPositiveLengthMeasure
lLength of each weld element.
REFERENCE  Symbol l according to ISO 2553:2019.
NumberOfWeldElements P_SINGLEVALUE / IfcCountMeasure
nNumber of weld elements.
REFERENCE  Symbol n according to ISO 2553:2019.
DeepPenetrationThroatThickness P_SINGLEVALUE / IfcPositiveLengthMeasure
sNominal throat thickness or effective throat thickness to which a certain amount of fusion penetration is added.
REFERENCE  Symbol s according to ISO 2553:2019.
WeldLegLength P_SINGLEVALUE / IfcPositiveLengthMeasure
zDistance from the actual or projected intersection of the fusion faces and the toe of a fillet weld, measured across the fusion face.
REFERENCE  Symbol z according to ISO 2553:2019.
Intermittent P_SINGLEVALUE / IfcBoolean
IntermittentIf fillet weld, intermittent or not
Staggered P_SINGLEVALUE / IfcBoolean
StaggeredIf intermittent weld, staggered or not

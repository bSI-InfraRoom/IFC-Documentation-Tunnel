7.8.4.136 Pset_TicketVendingMachine

PSET_TYPEDRIVENOVERRIDE / IfcElectricAppliance / VENDINGMACHINE

Properties of ticket vending machine. The property set can be used by IfcElectricAppliance with PredefinedType VENDINGMACHINE.
Properties
NameTypeDescription
TicketStuckRatio P_SINGLEVALUE / IfcRatioMeasure Indicates the ratio of tickets being stuck or jammed in the appliance.
MoneyStuckRatio P_SINGLEVALUE / IfcRatioMeasure Indicates the ratio of money being stuck or jammed in appliance.
PaymentMethod P_ENUMERATEDVALUE / IfcLabel / PEnum_PaymentMethod Indicates the vending machine payment method.
TicketProductionSpeed P_SINGLEVALUE / IfcIntegerCountRateMeasure Indicates the production speed of the ticket. It is measured by counting the number of tickets that can be produced per hour.
TicketVendingMachineType P_ENUMERATEDVALUE / IfcLabel / PEnum_TicketVendingMachineType Indicates the type of ticket vending machine.
VendingMachineUserInterface P_ENUMERATEDVALUE / IfcLabel / PEnum_VendingMachineUserInterface Indicates the type of vending machine user interface.

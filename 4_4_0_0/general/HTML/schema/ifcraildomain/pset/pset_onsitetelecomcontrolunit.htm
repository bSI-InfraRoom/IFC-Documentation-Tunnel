7.8.4.98 Pset_OnSiteTelecomControlUnit

PSET_TYPEDRIVENOVERRIDE / IfcController

Properties for on-site telecom control unit used for railway.
Properties

buildingSMART Data Dictionary

PSD-XML

NameTypeDescription
HasEarthquakeAlarm P_SINGLEVALUE / IfcBoolean Indicates whether the on-site control unit includes earthquake alarm function.
HasEarthquakeCollection P_SINGLEVALUE / IfcBoolean Indicates whether the on-site control unit collects earthquake information.
HasForeignObjectCollection P_SINGLEVALUE / IfcBoolean Indicates whether the on-site control unit collects foreign object information.
ControllerInterfaceType P_ENUMERATEDVALUE / IfcLabel / PEnum_ControllerInterfaceType Indicates the type of serial interface used by the device.
HasOutputFunction P_SINGLEVALUE / IfcBoolean Indicates whether the on-site control unit includes an output function.
HasRainCollection P_SINGLEVALUE / IfcBoolean Indicates whether the on-site control unit collects information on rain.
HasSnowCollection P_SINGLEVALUE / IfcBoolean Indicates whether the on-site control unit collects information on snow depth.
HasWindCollection P_SINGLEVALUE / IfcBoolean Indicates whether the on-site control unit collects information on wind.

7.8.4.17 Pset_CableSegmentTypeOpticalCableSegment

PSET_TYPEDRIVENOVERRIDE / IfcCableSegment / OPTICALCABLESEGMENT

Properties of optical cables segments. This property set is applicable to a type or occurence of IfcCableSegment with predefined type OPTICALCABLESEGMENT.
Properties
NameTypeDescription
NumberOfFibers P_SINGLEVALUE / IfcCountMeasure Indicates the number of fibers in the single tube or cable.
OpticalCableStructure P_ENUMERATEDVALUE / IfcLabel / PEnum_OpticalCableStructureType Distinguishes between different structures of an optical fiber cable.
NumberOfMultiModeFibers P_SINGLEVALUE / IfcCountMeasure Total number of multi-mode fibers in the optical fiber cable.
NumberOfSingleModeFibers P_SINGLEVALUE / IfcCountMeasure Total number of single-mode fibers in the optical fiber cable.
NumberOfTubes P_SINGLEVALUE / IfcCountMeasure Number of fiber tubes.
FiberMode P_ENUMERATEDVALUE / IfcLabel / PEnum_FiberMode Indicates the fiber mode.

7.8.4.5 Pset_BuiltSystemRailwayLine

PSET_TYPEDRIVENOVERRIDE / IfcBuiltSystem / RAILWAYLINE

Properties common to the definition of a railway line system, which is a set of functional tracks with explicit terminals. It is usually composed of a set of tracks with continuous track parts and alignments.
Properties
NameTypeDescription
LineID P_SINGLEVALUE / IfcIdentifier The unique identifier of the line.
IsElectrified P_SINGLEVALUE / IfcBoolean Indicates whether the track system is electrified or not.
LineCharacteristic P_ENUMERATEDVALUE / IfcLabel / PEnum_LineCharacteristic Indicates the characteristic of the line.

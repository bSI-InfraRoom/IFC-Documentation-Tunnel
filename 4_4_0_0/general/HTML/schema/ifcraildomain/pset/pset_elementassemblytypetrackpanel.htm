7.8.4.67 Pset_ElementAssemblyTypeTrackPanel

PSET_TYPEDRIVENOVERRIDE / IfcElementAssembly / TRACKPANEL

Track panel properties used in railway. The property set can be used by the predefined type TRACK_PANEL of IfcElementAssembly.
Properties
NameTypeDescription
IsAccessibleByVehicle P_SINGLEVALUE / IfcBoolean Indicates whether the element is accessible by a vehicle or not.
TrackExpansion P_BOUNDEDVALUE / IfcPositiveLengthMeasure In curvature context, bounded value of the expansion distance that can be added to rail gauge.
InstallationPlan P_REFERENCEVALUE / IfcDocumentReference / IfcValue Reference to external information source about installation or construction plan of the element.
